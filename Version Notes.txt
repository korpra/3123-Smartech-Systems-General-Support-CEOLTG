1.1.16	2020-05-05 fjm
-Use the Spec file switch IsMarked to identify trimmed sheets.  Then QA region location changes based on trimmed and untrimmed sheets.
	Trimmed (IsMarked = TRUE) 
        VBSOffset = Math.Round((MeasureedWidth - ProductWidth) / 2)
        QAStart = VBSOffset + TrimCompression
        QAEnd = QualityWidth + QAStart
	Untrimmed (IsMarked = FALSE)
		VBSOffset = 0
	    QAStart = (ProductWidth - QualityWidth * 25.4) / 2
        QAEnd = QualityWidth + QAStart
